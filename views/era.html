Общая архитектура системы

Система построена на основе микросервисной архитектуры. Она содержит в себе 8 микросервисов. На данном этапе, полностью реализовано только 5, из которых 3 было реализовано мной.

Архитектура микросервиса

Проектирование архитектуры каждого их микросервисов производилось на основе архитектурного подхода «Clean Architecture».

Микросервис «Авторизация»

Этот микросервис отвечает за авторизацию всех пользователей в системе. Авторизация построена на основе JWT токенов.

Микросервис имеет две конечные точки: для авторизации и получения авторизационных токенов, и для обновления этих токенов.

Для формирования токенов, микросервис производит обращения к микросервисам «Пользователи», «ВУЗ» по протоколу gRPC.

Микросервис «Пользователи»

Микросервис «Пользователи» выполняет функции хранения информации о всех пользователях в системе.

Было реализовано 5 типов пользователей: студент, преподаватель, диспетчер факультета, администратор ВУЗа и суперпользователь.

Для некоторых действий в системе были реализованы функции подтверждения действий средствами отправки токенов подтверждения на электронную почту.

Для хранения токенов использовалось хранилище Redis, а их отправка производилась по протоколу SMTP.

Микросервис «Курсы»

Микросервис курсы служит для организации образовательных курсов.

API шлюз

В рамках данного проекта, также был реализован API шлюз, который отвечает за маршрутизацию запосов к микросервисам и валидацию авторизационных токенов.

Библиотека «gcobs»

Библиотека «gcobs» решает проблему непредвиденных разрывов соединений между микросервисами. В ее задачи входит наблюдение за соединением и воспроизведение попыток восстановить соединение.

Исходный код библиотеки: gitflic.ru/project/education/gcobs.

Библиотека «errors»

Простая библиотека для создания типов ошибок и взаимодействия с ними.

Библиотека позволяет «вкладывать» ошибки друг в друга, проверять тип ошибки и наличие ошибки определенного типа в стеке ошибок.

Для некоторых функций библиотеки были написаны табличные unit-тесты.

Исходный код библиотеки: gitflic.ru/project/education/errors.
